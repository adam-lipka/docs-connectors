= ServiceNow Connector - Mule 4

Support Category: Select

The ServiceNow connector facilitates connections between Mule and ServiceNow apps. You can use the ServiceNow operations with the custom ServiceNow tables, along with any operations available through the installed plugins.

== Audience

* New user -- Read this document and configure your Mule application in xref:servicenow-connector-design-center.adoc[ServiceNow - Design Center].
* Current Mule user -- Read this document and configure your Mule application in xref:servicenow-connector-design-center.adoc[ServiceNow - [Design Center] or xref:servicenow-connector-studio.adoc[ServiceNow - Studio].
The xref:servicenow-connector-config-topics.adoc[ServiceNow - Configuration Topics] help you extend your understanding of Salesforce actions.
* Power user - Read the xref:servicenow-connector-xml-maven.adoc[ServiceNow - XML and Maven Support], xref:servicenow-connector-config-topics.adoc[ServiceNow - Configuration Topics], and xref:servicenow-connector-use-cases.adoc[ServiceNow - Use Cases].

== Prerequisites

This document assumes that you are familiar with ServiceNow, HTTP, SOAP, REST, Bulk and Streaming APIs, Mule runtime engine (Mule), Anypoint Connectors, Anypoint Studio (Studio), Mule concepts, elements in a Mule flow, and Global Elements.

== Develop an application

. Configure the connector.
. Test the connection.
. Build the rest of your flow.
. Add and configure DataWeave.

[[servicenow-actions]]
== ServiceNow Actions

The following list contains some of the actions you can perform using the ServiceNow connector:

* Aggregate
* Delete Multiple
* Delete Record
* Get
* Get Keys
* Get Records
* Insert
* Insert Multiple
* Update

Note: The ServiceNow SOAP site provides a full list of actions for any version of the ServiceNow connector.

== Install Zuora

. In Anypoint Studio, click the Exchange icon in the Studio task bar.
. Click Login in Anypoint Exchange and supply your Anypoint Platform username and password.
. Search for the Zuora connector and click Install.
. Follow the prompts to install the connector.

== Configure this Connector

To configure:

. Drag and drop a connector operation to the Studio canvas.
. Click the operation.
. Click the green plus sign to the right of *Module Configuration*.
. Set the values for authentication:
+
** basePath: Enter the base path of the REST api for your instance. Usually it is the `/rest` path.
** username: Enter the Zuora username.
** password: Enter the corresponding password.
** host: Enter the path to the Zuora API, for example, the `apisandbox-api.zuora.com` host.
** *port* - The port number the proxy server runs on.
** *protocol* - The username to log in to the server.

 After setting the *Global Element Properties* screen, return to the main connector menu and set the <<Connector Property Values>>.
